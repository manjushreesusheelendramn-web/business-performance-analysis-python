A Python project that analyzes monthly sales, expenses, and profit trends using Pandas, NumPy, and Matplotlib to visualize business performance over time.

# Business Sales, Expense and Profit Trend Analysis Using Python

This project analyzes a company's sales, expenses, and profit over a two-year period using Python. It calculates important financial metrics such as profit, sales growth, total profit, and maximum profit, then visualizes the results with graphs to make business trends easier to understand.

## Features

- Analyzes monthly sales and expense data
- Calculates monthly profit
- Calculates sales growth percentage
- Finds total and maximum profit
- Visualizes sales, expenses, and profit trends using graphs

## Built With

- Python 3
- Pandas
- NumPy
- Matplotlib
- Datetime
